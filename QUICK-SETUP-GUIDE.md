# 🚀 Quick Setup: Add Ventures to Your Database

## Step-by-Step Instructions

### 1. Open Your Notion Workspace
Go to: **TE COURSEWORK Y3** page

### 2. Find the "Undertakings Database"
Scroll down to find the database view (currently empty)

### 3. Click "New" to Add Each Venture
For each venture below, click **"New"** and fill in:

| Field | What to Enter |
|-------|---------------|
| **Code** | The venture code (V01, V02, etc.) |
| **Title** | The full venture name |
| **Type** | Select: Competency / Learning by Doing / Learning for Others |
| **Competency Focus** | Select relevant competencies |
| **Start Date** | Enter the start date |
| **End Date** | Enter end date or leave blank for ongoing |
| **Notes** | Paste the venture details |

### 4. Add All 22 Ventures

Copy each venture's data from the guide file and add them one by one.
