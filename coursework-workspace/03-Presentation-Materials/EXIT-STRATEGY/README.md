## Quick Summary

**Business:** AI Infrastructure for UK SMEs
**Model:** £1K setup + £200/mo retainer + 10% operational margin
**Traction:** 5 clients in pipeline, £1K paid project delivered
**Exit:** Trade sale (£150-250K), Bootstrap (£500K+), or License IP

## Key Shift

Transitioning from code development (one-off projects) to AI infrastructure (recurring revenue). This makes the business more valuable for exit because:
- Recurring revenue > project-based
- Scalable without founder's time
- Infrastructure = IP that can be licensed
