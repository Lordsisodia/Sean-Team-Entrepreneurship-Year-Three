# Presentation Slides

Each slide is in its own HTML file for easy editing:

| File | Slide |
|------|-------|
| `01-title.html` | Title slide |
| `02-where-ive-been.html` | Where I've Been (journey) |
| `03-exit-strategy.html` | Exit Strategy Overview |

## To add more slides:

1. Create new file: `04-slide-name.html`
2. Copy the structure from existing slides
3. Link `shared.css` for consistent styling

## To combine into one presentation:

Edit `presentation.html` to import each slide as a section, or open each file separately in the browser.

## Shared CSS

All slides use `shared.css` for consistent styling (colors, fonts, timeline, cards, etc.)

## Running locally

Open any `.html` file in a browser to preview that slide.
