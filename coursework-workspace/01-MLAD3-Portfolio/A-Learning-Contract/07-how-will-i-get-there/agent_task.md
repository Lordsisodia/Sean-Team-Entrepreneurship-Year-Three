### Writing Strategy

**Principles:**
1. **Values-first** - Each competency starts with the values/ethical dimension
2. **Specific situations** - Each competency includes a concrete team situation
3. **Personal reaction** - Each competency includes "I felt..." statement
4. **Honest assessment** - Maintain the candid self-reflection style
5. **Concise expression** - Ruthless editing to hit 500 words
6. **Active voice** - "I will", "I am", not passive constructions

**Template for each competency:**
```
**[Competency Name]** ([Area]: [Current]/10 → [Target]/10)

The core values challenge here is [values dimension]. I've struggled with [honest current level] - [specific situation with team member]. I felt [personal reaction], which showed me [insight].

To reach [target level], I will [specific action]. This matters because [connection to exit strategy]. I'll measure progress by [specific metric], with [timeline].
```
