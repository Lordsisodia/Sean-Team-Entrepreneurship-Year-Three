## 📝 WRITING STRATEGY PER COMPETENCY

### **The "Outstanding Formula" for Each Competency**

Based on first principles and marking criteria, each undertaking needs:

#### **Section 1: Context (50-75 words)**
```
START WITH YEAR 2 BASELINE
→ "In Year 2, I identified [competency] as [level/challenge]..."

CONNECT TO YEAR 3 OBJECTIVE
→ "For Year 3, this is critical because [connection to SMART objective]..."

STATE THE PROGRESSION
→ "I aimed to develop from [Year 2 level] to [Year 3 level], measured by [success metric]..."
```

#### **Section 2: Action Taken (150-200 words)**
```
SPECIFIC ACTIVITIES (not vague)
→ "I developed [competency] through [3-4 specific activities]: (1) [Activity with details], (2) [Activity with details], (3) [Activity with details]..."

TIMELINE + PEOPLE + FRAMEWORKS
→ "Timeline: [when]. People involved: [Sam, team, mentors]. Frameworks applied: [specific models/theories]..."

EVIDENCE INTEGRATION
→ "For example, [specific story with date and outcome]..."
```

#### **Section 3: Critical Evaluation (200-250 words) - MOST IMPORTANT**
```
THE HARDEST PART (honesty)
→ "The hardest part was [specific challenge]. I expected [expectation] but [reality]..."

WHAT SURPRISED YOU
→ "The most surprising insight: [unexpected learning]. This contradicted my belief that [previous assumption]..."

MISTAKES + WHAT YOU'D DO DIFFERENTLY
→ "What I'd do differently: [specific changes]. If starting again, I would [different approach]..."

CONNECTION TO YEAR 2
→ "This connects to Year 2's [challenge] by [how you evolved]..."
```

#### **Section 4: Learning Achieved (100-125 words)**
```
SPECIFIC SKILLS
→ "I developed [specific skills]: [skill 1], [skill 2], [skill 3]..."

MEASURABLE PROGRESS
→ "Before: [Year 2 baseline]. After: [Year 3 achievement with numbers]..."

CONNECTION TO SMART OBJECTIVE
→ "This directly connects to [SMART Objective] because [connection]..."
```

#### **Section 5: Future Application (50-75 words)**
```
NEXT MILESTONE
→ "Continue developing by [specific next steps]. Target [next milestone] by [when]..."

REMAINING GAPS
→ "Remaining gaps: [honest assessment]. Next focus: [specific area]..."
```
