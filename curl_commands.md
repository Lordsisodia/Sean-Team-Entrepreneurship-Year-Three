# Add Ventures to Notion - Quick Instructions

## Option 1: Python Script (Recommended - Fastest)

1. **Get your Notion Integration Token:**
   - Go to: https://www.notion.so/my-integrations
   - Click "New integration"
   - Give it a name like "Ventures Importer"
   - Copy the "Internal Integration Token" (starts with `secret_`)

2. **Edit the Python script:**
   ```bash
   nano add_ventures_to_notion.py
   ```
   - Replace `YOUR_NOTION_INTEGRATION_TOKEN_HERE` with your token
   - Save: Ctrl+O, Enter, Ctrl+X

3. **Run the script:**
   ```bash
   python3 add_ventures_to_notion.py
   ```

That's it! All 22 ventures will be added automatically. ⚡
